Repository for **Universal Basic Income as a Socio-Economic Tool of Contemporary States** research work.

## Datasets

* Create the following file structure:

```
root
│
├───Area
│   │
│   ├───Region
│   │   ├───indicator.csv
│   │   └─── ...
│   │
│   └───Region
│       ├───indicator.xlsx
│       └─── ...
```

* `csv` tables should be created in the next view:

```
Year;Indicator
year;value
year;value
...
```
